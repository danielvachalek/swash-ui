## Getting Started

### 1 - Install

```
npm install
```

### 2 - Fix

```
npm audit fix
```

### 3 - Start

```
npm start
```

### 4 - Build

```
npm run build
```

```
npm run migrate
```

you can use 
```
npm run migrate -- --dst=../firefox/dashboard2
``` 

if you want to change dst folder or src.
